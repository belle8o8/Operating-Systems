Starting up your computer and/or device is one operating system function.  
If your device isn't on, then it does nothing.  
You have to give the device instructions on what it needs to do and having it on is the first step.

Shutting down your computer/and or device is another operating system function.
There are different ways on shutting down your computer and/or device.
One way is putting your computer on sleep, which puts in low battery mode and saves your work.

Another operating system function is user interface which controls how you enter data and how it is displayed.
There are two types of user interfaces which are graphical and command line.
Graphical user interface is controlled with using menus.
Command line interface is using abbreviations or special key combinations on keyboard.
